# rmstnma Manuscript

This directory contains the manuscript draft and supporting files for the `rmstnma` project.

## Files

- `manuscript.Rmd`: The main manuscript draft.
- `references.bib`: BibTeX bibliography file.

## Usage

To render the manuscript, ensure you have the `rmstnma` package installed (and CmdStan configured if you wish to run the code chunks).

```r
rmarkdown::render("manuscript.Rmd")
```
